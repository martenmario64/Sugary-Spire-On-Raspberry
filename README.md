# What Is Sugary Spire On Raspberry Project?
This is a fan-made installer for Sugary Spire on that can be played On Standard Raspberry Spire OS (Heh Get It?), Ubuntu,

And Other ARM-Based Processors With A 64-Bit OS.

# Mininum System Requirements
A Raspberry Pi 5 (8GB Of Ram Or Above)

An Official Power Supply (Like The Ones From Canakit)

(OPTIONAL) Pironman5 (Not Needed, But Recommended.)

#How To Install:
Get the SugarySpireSetup.sh File In Releases.

# Troubleshooting

If It Does Not Prompt You For A 4K Kernel And It Crashes, Place "kernel=kernel8.img" at the end of
"/boot/firmare/config.txt".

If The Install Failed And You Wanna Redo The Main Install, Go To A hidden Folder In The Home Folder Called
$HOME/.martenapps/sugardemo/ and $HOME/.martenapps/sugardemo45/
